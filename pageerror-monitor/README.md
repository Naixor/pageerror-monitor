# Page error monitor

## Install

```shell
$ npm install -g pageerror-monitor
```

## Useage

> run
```shell
pageerror -h
```
or
```shell
pageerror --useage
```

You can also see `./examples/config.json` and `./examples/url.txt`.

And then you can run
```shell
pageerror -u {pathto url.txt} -c {pathto config.json}
```
